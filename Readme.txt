Не пугайся странных комментариев (это то, что с # Какой-то текст) на английском. Это мои комментарии, которые нужны лично мне по большей части.
Они мне тупо помогаю ориентировать в своём же коде не читая его.

Про токен и id_группы.
Есть файл que_cnf.py - тут храняться все статичные переменные в том, числе и токен и id группы.
Я там подписал их, но думаю и понять просто:
bot_token - это токен группы. Писать его после = и обязательно в кавычках (не важно " или ')
club_id - это id группы. Найти можно открыв свою группы в ВК, потом Управление, и под "Адрес" есть текст "Номер сообщества", вот он и нужен, но только цифры. Писать его после = и НЕ в кавычках, а прям числом.

Где менять админов?
id всех админов храняться в конфиг файле (que_cnf.py) в переменной white_list.
Чтобы добавить его нужно внутри [ ] после самого последнего id поставить запятую и вставить числом id-шник.
ВАЖНО: Брать только цифровой id и никакой более! Например, pe4enkas НЕ подходит.
Чтобы убрать админа достаточно стереть сам id и оставшуюся с ним запятую.

Про файлы.
que - основной, кроме того, что русский текст трогать НЕ рекомендую
que_ cmd - это файл функций и обработчиков, кроме того, что русский текст трогать НЕ рекомендую
que_cnf - это конфиг файл, кроме того, что русский текст трогать НЕ рекомендую

Вроде всё. Консультации по настройке бесплатны.
